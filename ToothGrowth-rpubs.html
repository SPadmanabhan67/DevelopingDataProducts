Overview

ToothGrowth Data Explorer is an interactive application that allows data exploration, analysis, modeling of ToothGrowth Data set available on Cran-R site.

The product allows user to:

  • Interactively Browse ToothGrowth dataset with search and pagination options on ToothGrowth
  • Analyze and visualize impact of dosage levels and supplements on tooth growth
  • Visualize a simple linear regression model for prediction.
